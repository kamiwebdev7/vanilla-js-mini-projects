# Modal & Menu Slider (Vanilla JavaScript)

A responsive landing page featuring a sliding side navigation menu and a modal popup, built using pure HTML, CSS, and JavaScript.

## Features
- Sliding side navigation menu
- Modal popup with smooth animation
- Click outside to close modal and menu
- Keyboard accessibility (ESC key)
- Clean and responsive layout

## Tech Stack
- HTML5
- CSS3 (Flexbox, animations, variables)
- Vanilla JavaScript (DOM manipulation, events)

## Use Case
This project demonstrates UI interaction logic without any frameworks, focusing on JavaScript fundamentals.
