# Vanilla JavaScript Mini Projects

A collection of small, focused projects built using **HTML, CSS, and pure JavaScript**.  
These projects demonstrate core frontend fundamentals such as **DOM manipulation, event handling, form validation, and data processing** without using any frameworks or libraries.

This repository is intended to showcase strong JavaScript basics and problem-solving skills.

---

## 📂 Projects Included

### 1️⃣ Modal & Menu Slider
**Folder:** `modal-menu-slider`

A responsive landing page featuring:
- Sliding side navigation menu
- Modal popup with animation
- Click-outside and keyboard (ESC) close functionality
- Clean UI and smooth transitions

**Concepts covered:**
- DOM manipulation
- Event handling
- UI state management using class toggling

---

### 2️⃣ Form Validator
**Folder:** `form-validator`

A client-side form validation project that provides real-time user feedback.

**Features:**
- Required field validation
- Username and password length validation
- Email format validation using regex
- Password confirmation matching
- Error and success UI states

**Concepts covered:**
- Form handling
- Reusable validation functions
- Regex usage
- UX-friendly error handling

---

### 3️⃣ DOM Array Methods
**Folder:** `dom-array-methods`

An interactive project demonstrating JavaScript array methods with real API data.

**Features:**
- Fetch random users from an external API
- Display dynamic data in the DOM
- Use of `map()`, `filter()`, `sort()`, and `reduce()`
- Calculate total wealth dynamically

**Concepts covered:**
- Async JavaScript (`fetch`, `async/await`)
- State management without frameworks
- Higher-order array methods
- Dynamic DOM rendering
