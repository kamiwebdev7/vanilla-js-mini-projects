# DOM Array Methods (Vanilla JavaScript)

An interactive project demonstrating JavaScript array methods and DOM manipulation using real data from an external API.

## Features
- Fetch random users from an API
- Dynamically render data to the DOM
- Double user wealth using `map()`
- Filter millionaires using `filter()`
- Sort users by wealth using `sort()`
- Calculate total wealth using `reduce()`

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## What I Learned
- Working with async JavaScript and APIs
- Managing application state without frameworks
- Using higher-order array methods
- Dynamically updating the DOM
