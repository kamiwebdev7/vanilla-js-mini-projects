# Form Validator (Vanilla JavaScript)

A client-side form validation project built using HTML, CSS, and pure JavaScript.  
It validates user input with clear visual feedback and error messages.

## Features
- Required field validation
- Username & password length validation
- Email format validation using regex
- Password confirmation matching
- Success & error UI states

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (DOM manipulation, events)

## What I Learned
- Handling form validation without libraries
- Reusable validation functions
- DOM traversal and UI state updates
- Writing clean, readable JavaScript
